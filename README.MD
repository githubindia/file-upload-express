# file-upload-express

An example of how to upload images locally with Node.js and Multer.

### Version
1.0.0

## Usage

Test to upload files locally using multer.

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser

```sh
$ npm start
```
OR

```sh
$ node index.js
```
